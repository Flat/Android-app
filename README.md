* Now Playing

 - Playback Control
 - Music Visualizer
 - Last Played and Queue mini lists (with gestures |<= O =>|)
* Notifications

 - Playback Control
* Searching/Requesting Songs
 - Search autocompletion
* Homescreen & Lockscreen Widget
* Full Last Played and Queue Lists
* Configurable Settings
